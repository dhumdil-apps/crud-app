# CRUD App

## Setup
```bash
# Install project dependecies:
$ npm i

# Configure Environment Variables (see .env.example)
```

## Development
```bash
# Watch both client and server files:
$ npm run start-client
$ npm run start-server
```

## Production
```bash
$ npm start:prod
```

## Utilities:
```bash
$ npm run generate-secret-key
```
